# Programa 3: Elección múltiple

## Objetivo

Aprender a usar las estructuras de control para establecer varias condiciones.

## Estructura del programa

Sentencia Switch...case, mostrar día de la semana a partir del número de día de la semana

## Resultado esperado

Si se selecciona 1 Lunes, si se selecciona 2 Martes, etc
