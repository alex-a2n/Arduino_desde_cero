\# Programa 5: Operador AND Lógico (\&\&)



\## Objetivo

Aprender a utilizar el operador AND (\&\&) para combinar condiciones.



\## Conceptos

\- Operador AND (\&\&)

\- Devuelve true solo si ambas condiciones son true

\- Tabla de verdad del AND



\## Resultado esperado

```

=== OPERADOR AND (\&\&) ===

(25 >= 18) \&\& (150 >= 100): 1

(25 >= 18) \&\& (150 >= 200): 0

```
