# Programa 7: Operador NOT Lógico (!)

## Objetivo
Aprender a utilizar el operador NOT (!) para invertir un valor booleano.

## Conceptos
- Operador NOT (!)
- Invierte true a false y viceversa
- Tabla de verdad del NOT

## Resultado esperado
```
=== OPERADOR NOT (!) ===
sistemaEncendido = 1
!sistemaEncendido = 0
```
