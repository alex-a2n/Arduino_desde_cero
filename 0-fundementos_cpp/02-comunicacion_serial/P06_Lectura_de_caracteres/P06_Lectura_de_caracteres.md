\# Programa 5: Lectura de un Carácter



\## Objetivo



Aprender a leer un carácter individual enviado desde el Monitor Serie usando `Serial.read()`.



\## Conceptos



\- Función `Serial.available()` para verificar datos

\- Función `Serial.read()` para leer un carácter

\- Lectura de un byte individual

\- Verificación antes de leer



\## Descripción



Este programa espera a que el usuario escriba un carácter en el Monitor Serie y lo muestra en la consola
